  # Bar Charts in Python Using Real News Data

  Learn to create bar charts in Python using Matplotlib and real news data from the NewsDataHub API.

  ## What You'll Build

  - **Topic distribution chart** — Top 15 most popular topics
  - **Language distribution chart** — Article distribution across languages
  - **Top 10 sources chart** — Most active news publishers
  - **Political leaning chart** — Bias distribution (optional, paid plan only)

  ## Quick Start

  **Option 1: Run with sample data (no API key needed)**
  ```bash
  pip install requests matplotlib
  jupyter notebook bar-charts-python-tutorial.ipynb
```

  Option 2: Run with live data
  1. Get free API key: https://newsdatahub.com/login
  2. Set your key in the notebook or Python script
  3. Run the code

  Files

  - bar-charts-python-tutorial.ipynb — Interactive Jupyter notebook (recommended)
  - bar_charts.py — Standalone Python script
  - data/sample-news-data.json — Sample dataset (200 articles)

  Output

  Generates 3-4 PNG files:
  - topic-distribution-chart.png
  - language-distribution-chart.png
  - top-sources-chart.png
  - political-leaning-chart.png (if data available)
